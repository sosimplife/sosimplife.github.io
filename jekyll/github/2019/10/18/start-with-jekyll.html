利用Jekyll在Github上搭建个人主页

Oct 18, 2019

Jekyll是一个用于创建静态网站的工具，用户编写的markdown文件转换成精美的Html文件。什么是静态网站呢？所谓静态网站是指所有的网站内容都是由Html文件组成，不会使用ASP/Javascript/PHP等语言，也不使用数据库。

Github Page是Github为每个账号提供的个人博客服务，支持静态网站。

1. Jekyll安装

本文的安装环境是Ubuntu 18.04.3 LTS。

Jekyll已经创建了一个Blog，将_post文件夹下的markdown文件改名为md文件。

打开浏览器，访问127.0.0.1：4000访问Blog。

2. Github Page设置

注册Github账号，启用Github Page，Github Page是一个特殊的代码仓库，其名字与Github账号名相同，向该仓库中上传网站代码，通过访问<用户名>.github.io域名即可浏览网站。

在_site目录下创建本地仓库，并将其推送到Github Page仓库。
